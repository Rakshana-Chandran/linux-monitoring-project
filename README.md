# Linux Server Monitoring Project

## Project Overview
This project monitors Linux server health using a Bash script.

## Features
- CPU Usage Monitoring
- Memory Usage Monitoring
- Disk Usage Monitoring
- Server Uptime Check
- Running Services Check
- Automated using Cron Job

## Technologies Used
- Linux (Ubuntu)
- Bash Scripting
- Cron
- Git
- GitHub

## How to Run

```bash
chmod +x monitor.sh
./monitor.sh
